#GPACalculator
使用C++语言开发的绩点计算工具

#用法
使用本程序前请先在data.txt中输入你的课程信息，格式如下：

[课程名称]空格[考试分数]空格[学分]    
例如：     
高等数学Ⅱ(上) 81 5.5     
大学体育（一）   89    2    
...    
    
每个科目占一行，空格个数不限制，程序会自动处理，但是要保证有以上三项，如果格式有误程序会自动告诉你修改。   
请勿修改文件名！   

本程序适用于成都理工大学的绩点计算规则，若规则有变，将及时更新。
